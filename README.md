# KPI一覧ダッシュボード

Googleスプレッドシートのデータを読み込んで、受講生の進捗（KGI/KPI/KDI、フェーズ別達成率、フォロワー数帯、投稿数など）を可視化する**単一HTMLのダッシュボード**です。

ビルド不要・サーバー不要。`index.html` をブラウザで開くだけで動作します（データ取得にネット接続が必要）。

---

## ファイル構成

```
.

├── DEPLOY_AWS.md     # AWS デプロイ手順（Amplify / S3+CloudFront 等）
└── .gitignore
```

外部ライブラリは CDN から読み込みます（リポジトリには含めません）:

- [PapaParse](https://www.papaparse.com/) — CSV/シートのパース
- [Chart.js](https://www.chartjs.org/) + chartjs-plugin-datalabels — グラフ描画
- [lz-string](https://github.com/pieroxy/lz-string) — 設定共有URLの圧縮
- Google Fonts（Noto Sans JP）

## 使い方

1. `index.html` をブラウザ（Chrome / Safari / Edge 等）で開く。
2. 初回起動時に **Google Sheets API キー** の入力を求められるので入力する。
   - 入力したキーは**そのブラウザの localStorage にのみ保存**され、ファイルには書き込まれません。
   - 次回以降は自動で読み込まれます。
3. スプレッドシートのデータが自動で読み込まれ、ダッシュボードが表示されます。
4. ヘッダー右上の「更新」でデータを再取得、「スプシ連携」で接続先シートを変更できます。

### データソース（スプレッドシート）

`index.html` 内の `DEFAULT_CATS` 定数に、接続先スプレッドシートのURLとシート名が定義されています。読み込むシート（カテゴリ）:

| キー | シート例 | 用途 |
|------|----------|------|
| `sp` | 投稿プログラム（旧） | SP/AP/PP ステップ完了率 |
| `post` | 投稿数 / フォロワー数 | 投稿数分析・フォロワー数帯 |
| `mansoku` | 全体のマンスリー数値 | 万垢率（KGI/KPI①） |
| `meibo` | 万垢達成者 | 達成者名簿 |
| `new_pp` | 投稿プログラム（新）| NPP（新カリキュラム）完了率 |

接続先を変える場合は `index.html` の `DEFAULT_CATS` を編集するか、画面の「スプシ連携」から設定します。

## デプロイ（Netlify）

静的ファイル1枚なのでそのまま公開できます。

- **Netlify Drop**: <https://app.netlify.com/drop> に `index.html` をドラッグ&ドロップ。
- もしくは このGitHubリポジトリを Netlify に連携（ビルドコマンド不要 / 公開ディレクトリ = リポジトリルート）。

### AWS で公開する場合

単一の静的HTMLなので AWS にもそのまま載せられます。**Amplify Hosting（GitHub連携・自動デプロイ）** が手軽でおすすめ。S3+CloudFront 等の手順も含め、詳細は **[DEPLOY_AWS.md](./DEPLOY_AWS.md)** を参照。

公開後、利用者は初回にAPIキーを入力すれば、以降は自動でダッシュボードが表示されます。

## セキュリティ上の注意

- **APIキーはファイルに埋め込まれていません。** 各利用者がブラウザで入力し、その端末にのみ保存されます。社内の信頼できるメンバーにのみキーを共有してください。
- Google Cloud Console で、このAPIキーに **「特定サイト（公開ドメイン）のみ」「Google Sheets API のみ」** の制限をかけることを推奨します。
- 公開リンクを知る人は受講生の個人情報（名前・フォロワー数・コース・クラス・担当MG等）を閲覧できます。一般公開ではなく**関係者限定の共有**を推奨します。データは**読み取り専用**で、スプレッドシート自体は書き換えられません。

## 補足

- データの読み込み・集計・グラフ描画はすべて**ブラウザ内**で完結します（バックエンドなし）。
- 卒業月（コホート）切り替え、フェーズ別表示、名前一覧モーダル、CSV手動アップロードなどに対応しています。
